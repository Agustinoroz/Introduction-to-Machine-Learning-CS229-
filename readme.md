Este es un curso muy famoso que imparte el profesor Andrew NG bautizado como introducción a Machine Learning. Promete que luego de algunas semanas de clase y tareas, se consigue una buena base para emprender algún proyecto propio que sea interesante y que tenga como objetivo investigar algun resultado a traves del aprendizaje automatizado.
Las clases de Andrew se imparten en la universidad de Stanford, sin embargo, el curso dictado en el año 2018 es público y está subido a YouTube para que todo el mundo pueda seguirlo.

He seguido esas clases y completado las tareas que Andrew propuso cada semana. En este respositorio se encuentran esas tareas junto a mis resoluciones y, en un futuro, estará el proyecto personal que espero poder comenzar.
